Project Description
FDFToolkit .NET makes it easier to read, write, create, populate, parse, merge FDF, XDP, XFDF, and XML data with Acrobat and LiveCycle PDF forms. FDFToolkit.net utilizes iTextSharp 4.x technologies, under MPL license.

FDFToolkit.net utilizes iTextSharp Technologies, under MPL license.
http://en.wikipedia.org/wiki/MozillaPublicLicense

A modified iTextSharp library source code can be found at:
http://www.nk-inc.com/downloads/itextsharp_4.0.7-source.zip

The iTextSharp 4.0.7 binary is included in the source code.

For support, examples, and related information please visit:
http://www.fdftoolkit.net/
